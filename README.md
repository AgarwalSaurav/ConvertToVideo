# ConvertToVideo
Convert slides in PDF to video

Author: Saurav Agarwal

License: GPL v3.0 

Requires: Linux system with `ffmpeg`, `imagemagick`, and `python3`

Command: 

`bash ConvertToVideo.bash input/input_timings -d input/ output.mp4`

`python3 slides2video.py sample_slides.pdf slides_timings sample_output.mp4`
